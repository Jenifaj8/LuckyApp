# 🎲 Lucky App

Lucky App is an ASP.NET Core MVC web application that allows users to explore number patterns and generate combinations for different lottery-style games.

## 🚀 Features

- Multiple game modules:
    - Lotto 6/49
    - Lotto Max
    - Daily Grand
    - Pick 4
    - Keno
- Pattern-based number generation
- User input processing
- Clean UI with Bootstrap styling
- Separate pages for each game

## 🧠 How It Works

Users enter a set of numbers, and the app applies a pattern-based algorithm to generate a new set using alternating operations.

## 🛠️ Tech Stack

- ASP.NET Core MVC
- C#
- Razor Views
- Bootstrap
